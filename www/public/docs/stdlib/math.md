# Math (math)

Mathematical functions.

## Functions

### abs(x: num) -> num
Absolute value.

### sqrt(x: num) -> num
Square root.

### pow(base: num, exp: num) -> num
Power.

### floor(x: num) -> num
Round down.

### ceil(x: num) -> num
Round up.

### round(x: num) -> num
Round to nearest integer.

### min(a: num, b: num) -> num
Minimum of two numbers.

### max(a: num, b: num) -> num
Maximum of two numbers.

## Constants

### PI
Value of pi (3.14159...).

### E
Euler's number (2.71828...).
