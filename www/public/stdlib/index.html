loft Standard Library

Welcome! Here's everything you need to know about the loft standard library, filled with all the handy tools you'll need to build your next project.

Primitives

The basic building blocks of any loft program:

str
Work with text, split strings, and format data.
Array
The go-to collection for lists of items.
num
Handle all your numbers and math.
bool
True or false, simple as that.
void
Represents the absence of a value.

Builtin Modules

Handy modules for all your day-to-day coding needs:

encoding
Encoding and decoding utilities for various formats
web
Web API for HTTP requests with comprehensive async support
math
Math builtin providing mathematical operations and constants
string
String utility functions
array
Array utility functions
object
Object manipulation utilities similar to JavaScript Object methods
fs
File system builtin providing file system operations
term
Terminal builtin for interacting with the terminal
random
Random number generation utilities
time
Time builtin for time-related operations
json
JSON parsing and stringification utilities
