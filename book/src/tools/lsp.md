# LSP and Editor Support

loft includes a Language Server Protocol implementation for IDE integration.

## Features

- Code completion
- Hover information
- Go to definition
- Find references
- Diagnostics
- Code formatting
- Signature help
- Semantic highlighting

## VSCode Extension

Install the loft extension from the VSCode marketplace for the best experience.

## Configuration

The LSP starts automatically when you open .lf files in supported editors.
