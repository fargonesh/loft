# Strings (string)

String manipulation utilities.

## Functions

### len(s: str) -> num
Get string length.

### split(s: str, delimiter: str) -> array
Split string by delimiter.

### trim(s: str) -> str
Remove whitespace from both ends.

### to_upper(s: str) -> str
Convert to uppercase.

### to_lower(s: str) -> str
Convert to lowercase.
